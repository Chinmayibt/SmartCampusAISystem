How it works

  1. Timetable — Add named blocks for each weekday (e.g. lectures). Gaps between blocks are where study can fit.
  2. Settings — Set session length (minutes) and the hours you allow for study (e.g. 8:00–22:00). Suggestions use these values.
  3. Tasks — Create assignments and exams with due dates. Optional reminders; log minutes as you work.
  4. Dashboard — See overdue and upcoming items, suggested study slots for a chosen day, and enable browser notifications if you want.
  5. Progress — View completed-this-week counts, total logged minutes, and recent study logs.
  6. Focus — Run a Pomodoro-style timer in the browser (not synced to the server).
  7. Goals — Jot weekly targets; stored only in this browser.

Start the app with npm start in the project folder, then open the URL shown in the terminal (usually http://localhost:3000).
