About this app

SmartStudy Planner is a local prototype: your timetable, tasks, and preferences are stored in a JSON file on the machine running the server. There are no user accounts.

It helps you see when you are busy, what is due, and suggested free slots for study based on gaps in your weekly schedule.

Built with HTML, CSS, JavaScript, Node.js, and Express — intentionally minimal so you can extend it.
